# Learn Spring Boot

A modular Spring Boot application designed to demonstrate best practices in backend development, including user authentication (JWT), product management, order processing, and payment integration (Razorpay).

## Overview

This project is structured as a multi-module Spring Boot application to maintain clean code and separation of concerns.

## Key Features

- **User Authentication**: Secure JWT-based authentication and authorization.
- **Product Management**: CRUD operations for product inventory.
- **Order Management**: Robust order placement and processing.
- **Payment Integration**: Seamless payment handling using the Razorpay API.
- **API Documentation**: Automated OpenAPI/Swagger documentation.

## Tech Stack

- **Java**: 23
- **Framework**: Spring Boot 4.0.6
- **Database**: PostgreSQL
- **Security**: Spring Security + JWT
- **Build Tool**: Maven
